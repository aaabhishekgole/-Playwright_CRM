# Playwright AI MCP Tutor

Use this framework skill when adding or updating tests inside `Advance-Playwright-Framework-Testing`.

## Rules

- Add locators only inside `src/pages`
- Put business orchestration in `src/modules`
- Keep specs under `src/tests`
- Use fixture imports from `src/fixtures`
- Use `test.step()` for every scenario
- Keep data in `src/testdata`
